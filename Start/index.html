About

本會一直致力為流浪貓狗提供糧食、醫療、絕育及領養等服務。除了主動援救流浪街頭的受傷貓狗外，本會亦會接收被市民遺棄的小動物，並盡力為牠們尋覓可信的主人覓尋一個永遠的家。

正如本會口號一樣，我們相信「支持領養等於拯救生命」。希望藉著設立領養部，給貓狗們一個永遠的家。由於領養工作需要人手眾多，有賴多年來多位義工熱心幫忙，協助暫托及家訪等事務，截至2020年，已成功為數千隻貓狗尋找到幸福家園。

「支持領養等於拯救生命」
藉著拯救可憐流浪貓狗及安排適合領 養人領養，防止牠們遭虐待或傷害。

Purpose

1.拯救來自繁殖埸及被無良主人遺棄的可憐流浪貓狗，防止牠們遭虐待或傷害。
2.在可運用的能力、人力以及財力下盡力拯救流浪貓狗。
3.以開放態度與其他行內動物組織合作。
4.盡量為拯救的貓狗覓得最好、最能信任的主人。
5.所有待領養貓狗必須家訪以確保將來聯絡及清楚確定居住地點適合。
6.盡力多做區內動物生育控制計劃，為香港流浪貓狗謀福祉。
7.致力推動領養訊息。
8.致力推動防止虐畜訊息。

Main

本會主要透過營運貓／狗舍及領養部，進行貓狗隻領域護理計劃、動物生育控制計劃以及從不同渠道宣揚動保信息以幫助流浪動物，為舒緩香港流浪動物問題出力。本會共有數十名在職義工，各施展不同才能協助本會。

貓舍、狗舍及領養部
本會貓舍及狗舍分別於2021年成立，為可憐貓狗提供安身之所。香港雖貴為一個國際大都會，但寵物被主人遺棄的事時有發生。棄養的籍口有很多，例如搬家、結婚、懷孕等，但寵物的結局只得一個：失去容身之所（就是失去了家）。這些動物有的被送往人道毀滅、有的淪落街頭，自生自滅。除飼主棄養外，繁殖場丟棄貓狗的問題亦相當嚴重。繁殖場的貓狗往往是利慾薰心的人類的牟利工具。牠們長年被困在衛生環境非常惡劣及極度狹小的空間，一生被迫不斷繁殖，直到生病及再沒有利用價值的時候，便會被繁殖場冷血無情地用完即棄。本會成立貓舍及狗舍，目的為協助這些流浪街頭、被遺棄或傷病的可憐貓狗得到好的治療和照顧，並尋找合適的領養人。

貓／狗舍主要服務範疇
。提供足夠及健康的糧食
。為傷病的貓狗隻提供治療和照顧
。為膽小害羞，怕人的貓給予適當訓練，以為被領養做好準備
。為年老的貓狗提供地方給予休養，好讓牠們安享晚年

正如本會口號一樣，我們相信「支持領養等於拯救生命」。希望藉著設立領養部，給貓狗們一個永遠的家。由於領養工作需要人手眾多，有賴多年來多位義工熱心幫忙，協助暫托及家訪等事務，截至2017年，已成功為大概2,300隻貓狗尋找到幸福家園。
另外，本會亦持開放態度與多個行內動物組織合作，如領養日、義賣會等，並全力推動宣傳領養替代買賣的意識，務求令小動物受到更多保護，更少傷害。雖然過程艱鉅，但當看見一隻隻流浪、受傷、無助的小動物得到適時的照顧，並找到一個幸福的家，我們都會非常欣喜雀躍，這也是我們一直堅持的其中一個原因。
